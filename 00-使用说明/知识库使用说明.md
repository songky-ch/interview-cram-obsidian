# 知识库使用说明

## 入库流程

1. 每天面试结束后, 将面试老师反馈原文记录到 `01-面试记录`。
2. 按技术栈拆分重点, 同步沉淀到 `03-知识点`。
3. 将可复用表达整理到 `04-优秀答案`。
4. 将答偏、答漏、概念混淆记录到 `02-错题卡片`。
5. 每周在 `05-能力周报` 汇总高频问题和补弱方向。

## 答案结构

优秀答案默认采用:

结论 -> 核心原理 -> 实践案例 -> 常见追问 -> 易错点
